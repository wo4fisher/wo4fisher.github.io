﻿﻿﻿多级菜单使用要点

##1 增加一级菜单新增内容：

1.1 在mainmenu增加菜单索引

g_con_list结构体新增按钮控件，能够切换到对应的窗口

1.2在gui_DlgMainMenuProc函数中增加新增对话窗口的open函数

###1.3在gui_Task()中增加响应的proc函数
###1.4 在gui.h中相关变量增加
####1.4.1 enum 枚举类型新增索引
enum { DLG_DESKTOP=1, DLG_MAINMENU,DLG_KEYDEMO,DLG_LEDDEMO,DLG_RTCDEMO,DLG_BUZZERDEMO,DLG_DHT11DEMO};
####1.4.2 OPEN函数和proc函数添加定义
####1.4.3 新增对话框结构体主要参数进行定义
dlg_key_demo_param，这一组时结构体组成的共用体，在对话框激活的时候初始化相关变量，不同的窗口可以有不同的结构体内部成员变量。

`union gui_all_dlg

{

struct dlg_desktop_param desktop_param;
struct dlg_key_demo_param key_demo;
struct dlg_led_demo_param led_demo;

　struct dlg_RTC_demo_param rtc_demo;
};`
